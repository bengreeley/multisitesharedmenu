Use a navigation menu from another multisite blog on the same network to achieve a universal common navigation area.

== Description ==
Allows users in a WordPress multisite network pull in a menu from another site in order to achieve a universal navigation or shared navigation without needing to manually recreate menus. 

Plugin is intended for WordPress Multisite environment for plugin to function properly and use the same theme to ensure .

== Installation ==
1. Copy the folder into your /wp-content/plugins directory.
1. Activate for the sites you wish to use plugin on. NOTE: Should not be activated on master menu site. 
1. Once activated, set up a menu to use as a “master”. 
1. To use this menu on other sites, browse to the site you wish to use the menu on. 
1. Navigate to ‘Shared Menu Settings’ under the ‘Appearance’ menu. 
1. Select the source site that contains the menu you wish to use and select the menu location that will use the menu.
1. Click ‘save changes’ to save your changes
